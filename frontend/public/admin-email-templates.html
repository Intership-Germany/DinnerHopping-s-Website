Email Templates

How to use variables

Use {{variable}} placeholders in your templates. Variables are replaced with actual values when emails are sent.

Automatic variables (available in all templates):
  • {{current_date}} - Current date (YYYY-MM-DD)
  • {{current_time}} - Current time (HH:MM:SS)
  • {{current_datetime}} - Full ISO datetime
  • {{current_year}} - Current year
Template-specific variables:

Document available variables in the "Variables" field (e.g., event_title, email, event_date)
